Detailed Description

Компонент расширения для подгрузки дополнительных полей трудозатрат из запроса сохранения WorkLogSettings

Загружает в настройки WorkLogSettings WorkLogSettings.UsersNotificationErrorAutoWorkLogIds WorkLogSettings.GroupNotificationErrorAutoWorkLogIds WorkLogSettings.OrganizationItemNotificationErrorAutoWorkLogIds WorkLogSettings.UsersListNotAffectLimitationWorkLogIds WorkLogSettings.GroupListNotAffectLimitationWorkLogIds WorkLogSettings.OrganizationItemListNotAffectLimitationWorkLogIds
