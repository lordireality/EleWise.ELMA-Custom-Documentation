◆ ScopeAttribute

static

Configuration attribute that can be passed to the M:EleWise.ELMA.CAB.IConfigurable.Configure(System.Collections.Specialized.NameValueCollection) implementation on the service, to determine the scope of the isolated storage to use for persistence. It must be a key in the dictionary with the name Scope.

Supported syntax is the same one as C#, except that the T:System.IO.IsolatedStorage.IsolatedStorageScope enum type must be omited:

Assembly | User

or:

Roaming | User | Assembly | Domain
