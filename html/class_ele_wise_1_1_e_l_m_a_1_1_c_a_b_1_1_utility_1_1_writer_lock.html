Detailed Description

Helper class that makes it easier to ensure proper usage of a T:System.Threading.ReaderWriterLock for writers by providing support for T:System.IDisposable and the using keyword.

Common usage is as follows:

ReaderWriterLock mylock = new ReaderWriterLock();
// Inside some method
using (new WriterLock(rwLock))
{
// Do your safe resource write accesses here.
}
