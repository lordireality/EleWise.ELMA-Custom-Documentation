Detailed Description

Indicates that the marked method builds string by format pattern and (optional) arguments. Parameter, which contains format string, should be given in constructor. The format string should be in M:System.String.Format(System.IFormatProvider,System.String,System.Object[])-like form.

[StringFormatMethod("message")]
public void ShowError(string message, params object[] args) { /* do something *&zwj;/ }
public void Foo() {
ShowError("Failed: {0}"); // Warning: Non-existing argument in format string
}
