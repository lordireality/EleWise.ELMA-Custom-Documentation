Detailed Description

Провайдер телефонии Asterisk
Должен сам заботиться о поддержании соединения (например, через реализацию ISweepHandler)
Может выдавать команды агенту (через реализацию IPingCommandsExtension)
Может реагировать на изменение настроек телефонии (через реализацию ITelephonySettingsEventHandler)
Может расширять модель настроек пользователя (IUserSettingsModel)
Может реагировать на изменение настроек пользователя (через реализацию IUserSettingsModelHandler)
