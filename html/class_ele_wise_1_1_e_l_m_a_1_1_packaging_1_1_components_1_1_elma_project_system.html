Получить относительный путь для размещения файла в файловой системе проекта

Для сборок, которые всегда размещаются в lib, срезается папка lib и папка платформы (если есть): lib\net45\Autofac.dll –> Autofac.dll Для прочих файлов путь не меняется: Content\index.html –> Content\index.html
