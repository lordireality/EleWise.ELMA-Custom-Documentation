inlinestatic

Узнать, отключено ли уведомление о просрочке срока завершения работ по задачам вложенного проекта

  • - Обязательные параметры Параметры: [0]*: Проектная задача (IProjectTaskBase)

Результат: True – В случае, если переданная задача является задачей вложенного проекта, для которого отключено оповещение В остальных случаях - False

Пример: Пусть ProjectTask - переменная типа "Базовый класс проектной задачи". Чтобы определить, является ли ProjectTask задачей вложенного проекта, для которого отключено оповещение: {DontNotifyStartWorkOnSubTasks({$ProjectTask})}

static FormatedValue EleWise.ELMA.Projects.Notifications.ProjectGeneratorFunctionContainer.IsProjectTaskDelayed ( FunctionEvaluationContext context)
inlinestatic

Узнать, является ли переданная задача отложенной, т.е. задача не входит в интервал отображения задач в проекте

  • - Обязательные параметры Параметры: [0]*: Проектная задача (IProjectTaskBase)

Результат: True – в случае, если переданная задача активна и не видима в списке пользователя, а так же при выполнении одного из условий:

  1. переданная задача является отложенной и для задачи не разрешен ранний старт
  2. Есть незавершенные связанные задачи, от которых зависит данная В остальных случаях - False

Пример: Пусть ProjectTask - Переменная типа "Проектная задача". Чтобы определить, является ли задача отложенной: {IsProjectTaskDelayed({$ProjectTask})}
