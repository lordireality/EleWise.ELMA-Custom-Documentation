Метка преобразования существующих данных

Атрибут устанавливается для описания метода в dbstructure.xml для пометки действия выполняемого только для преобразования существующей базы. В качестве значения рекомендуется указывать версию ELMA, для которой добавляется действие. Помеченное таким образом действие при первом старте системы выполняться не будет.
