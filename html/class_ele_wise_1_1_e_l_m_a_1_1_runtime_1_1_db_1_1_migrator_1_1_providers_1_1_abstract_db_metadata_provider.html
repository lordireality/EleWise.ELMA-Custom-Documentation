◆ ConvertToRegularDbIdentifier()

string EleWise.ELMA.Runtime.Db.Migrator.Providers.AbstractDbMetadataProvider.ConvertToRegularDbIdentifier ( string name)
abstractprotected

Привести идентификатор к обычному виду, который применяется в СУБД, когда он не совпадает с ключевым словом.

Например: в Oracle обычные идентификаторы указываются в Upper Case, в PostgreSQL - в Lower Case, MSSQL - As Is (как есть).
