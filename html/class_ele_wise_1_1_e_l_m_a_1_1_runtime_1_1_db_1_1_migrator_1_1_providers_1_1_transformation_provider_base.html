virtual IEnumerable< string > EleWise.ELMA.Runtime.Db.Migrator.Providers.TransformationProviderBase.ExcludedOldReservedWords ( )
inlineprotectedvirtual

Список старых зарезервированных слов, которые не нужно учитывать

Returns
Список зарезервированных слов

Связано с миграцией с одной субд на другую Если на предыдущей были зарезрвированные слова, которых нет в новой, и при этом обращение к ним выполняется до выполнения преобразования бд

Например, при миграции с MSSQL на PostgreSQL в списке зарезервированных слов сохраняется слово "Structure", которое является колонкой в таблице хранения текущей структуры БД. Обращение к ней производится до выполнения преобразования наименований таблиц и колонок, названных зарезервированным словом. При этом, ни в одной из версий PostgreSQL данное слово не является зарезервированным
