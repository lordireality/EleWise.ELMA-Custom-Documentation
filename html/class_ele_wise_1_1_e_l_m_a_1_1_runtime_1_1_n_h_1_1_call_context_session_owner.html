Detailed Description

Владелец сессии NHibernate на основе контекста вызова. Позволяет работать с сессией в рамкаж жизненного цикла данного объекта.

Поддерживается провайдерами T:EleWise.ELMA.Runtime.NH.SessionProviders.WebSessionProvider и T:EleWise.ELMA.Runtime.NH.SessionProviders.CallSessionProvider. Может использоваться внутри существующего владельца. В этом случае реальным владельцем сессии будет корневой объект.

Пример использования:

using (CallContextSessionOwner.Create())
{
var user = new User();
user.Name = "TestUser";
user.Save();
}
