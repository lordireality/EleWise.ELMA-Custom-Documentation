Detailed Description

Стратегия NHibernate получения getter-ов и setter-ов для свойств.

Для свойств дельта-сущностей (T:EleWise.ELMA.Model.Entities.EntityDynamic`1) используется внутренний механизм получения значения свойства.

Для свойств остальных классов используется рефлексия.

В констркуторе можно задать использовать или нет оптимизацию рефлексии (по умолчанию использовать): компилируемые методы для получения/установки значения свойства.

Для типа T:System.DateTime используется специальный getter, который значение F:System.DateTime.MinValue заменяет на null
