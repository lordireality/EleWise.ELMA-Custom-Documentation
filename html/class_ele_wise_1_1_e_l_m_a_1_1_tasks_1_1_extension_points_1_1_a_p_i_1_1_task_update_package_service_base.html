Detailed Description

Базовый класс для сервиса пакетов задач, нужен для того, чтобы в списке доступных наборов пакетов появилась понятная запись. Реалзиация этого сервиса не должна возращать никаких пакетов, все пакеты должны возращаться соответсвующим классом-реализацией T:EleWise.ELMA.Tasks.API.Service.v1.TaskBaseUpdatePackageService (с тем-же ServiceUid)
