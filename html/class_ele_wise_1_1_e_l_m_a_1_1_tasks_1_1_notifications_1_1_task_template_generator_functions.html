static FormatedValue EleWise.ELMA.Tasks.Notifications.TaskTemplateGeneratorFunctions.GetInformToExceptMembers ( FunctionEvaluationContext context)
inlinestatic

Формирует список пользователей из соисполнителей первой задачи, не являющихся соисполнителями во второй.

  • - Обязательный параметр Параметры: [0]*: Первая задача (ITaskBase) [1]*: Вторая задача (ITaskBase)

Результат: Возвращает список пользователей из соисполнителей первой задачи, не являющихся соисполнителями во второй

Пример: Пусть Task1, Task2 - переменные типа ITaskBase. В качестве соисполнителей Task1 указаны: "Петров Петр", "Сидоров Константин", а Task2: "Смирнов Денис", "Петров Петр"

Чтобы получить соисполнителей из Task1, не являющимися соисполнителями в Task2: {GetInformToExceptMembers({$Task1};{$Task2})} Результат: "Сидоров Константин"

И наоборот: {GetInformToExceptMembers({$Task2};{$Task1})} Результат: "Смирнов Денис"

Результат: Возвращает название типа сущности, к которой запрашивается доступ
