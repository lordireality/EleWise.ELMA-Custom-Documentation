static FormatedValue EleWise.ELMA.Tasks.Notifications.WorkLogItemGeneratorFunctionContainer.GetEntityDisplayNameByWorkLogItem ( FunctionEvaluationContext context)
inlinestatic

Получить отображаемое имя сущности.

  • - Обязательные параметры Параметры: [0]*: Запись в отчете о трудозатратах (IWorkLogItem)

Результат: Строка, содержащая отображаемое имя сущности

Пример: Пусть WorkLogItem - переменная типа "Запись в отчете о трудозатратах". В качестве объекта трудозатрат была выбрана задача Task. Чтобы получить отображаемое имя Task через WorkLogItem: {GetEntityDisplayNameByWorkLogItem({$WorkLogItem})}

inlinestatic

Вывести отформатированное значение интервала рабочего времени в днях, часах и минутах.

  • - Обязательные параметры Параметры: [0]*: Интервал рабочего времени (WorkTime)

Результат: Возвращает отформатированное значение интервала рабочего времени в днях, часах и минутах

Пример: Пусть WorkLogItem - переменная типа "Запись в отчете о трудозатратах". Чтобы вернуть значение фактического отработанного времени (FactWorkTime): {WorkTimeToFormatString({$WorkLogItem.FactWorkTime})}
