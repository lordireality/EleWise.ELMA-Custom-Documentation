Detailed Description

Проверка единственной привилегии Global || EntityType || Instance.

Для проверки привилегий типа Instance необходимо на метод контроллера повесить PermissionAtribute с конструктором по умолчанию + InstancePermissionAttribute на необходимый для проверки параметр метода
