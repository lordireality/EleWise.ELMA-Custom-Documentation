Detailed Description

Walks through the type hierarchy of data structure and builds code that handles some of entries (in imperative way).

Usage: 1) Call Run to walk trougth data structure. You can call Run in your own handlers - recursion is supported. 2) Call Compile to emit all walking functions and start code.
