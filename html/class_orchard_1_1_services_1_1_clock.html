IVolatileToken Orchard.Services.Clock.When ( TimeSpan duration)
inline

Provides a T:Orchard.Caching.IVolatileToken instance which can be used to cache some information for a specific duration.

Parameters
durationThe duration that the token must be valid.

This sample shows how to use the M:Orchard.Services.IClock.When(System.TimeSpan) method by returning the result of a method named LoadVotes(), which is computed every 10 minutes only.

_cacheManager.Get("votes",
ctx => {
ctx.Monitor(_clock.When(TimeSpan.FromMinutes(10)));
return LoadVotes();
});

IVolatileToken Orchard.Services.Clock.WhenUtc ( DateTime absoluteUtc)
inline

Provides a T:Orchard.Caching.IVolatileToken instance which can be used to cache some until a specific date and time.

Parameters
absoluteUtcThe date and time that the token must be valid until.

This sample shows how to use the M:Orchard.Services.IClock.WhenUtc(System.DateTime) method by returning the result of a method named LoadVotes(), which is computed once, and no more until the end of the year.

var endOfYear = _clock.UtcNow;
endOfYear.Month = 12;
endOfYear.Day = 31;
_cacheManager.Get("votes",
ctx => {
ctx.Monitor(_clock.WhenUtc(endOfYear));
return LoadVotes();
});
