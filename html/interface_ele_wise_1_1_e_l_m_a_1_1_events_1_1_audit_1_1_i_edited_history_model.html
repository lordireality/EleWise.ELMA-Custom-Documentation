Detailed Description

Модель отображения истории работы с объектом для действия "Изменение".

В данном случае не применяется к базовому действию F:EleWise.ELMA.Model.Actions.DefaultEntityActions.Update, для использования нужно добавлять в сущность собственное действие "Изменение" (Edit)
