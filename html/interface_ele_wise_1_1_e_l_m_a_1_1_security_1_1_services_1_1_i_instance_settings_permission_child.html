Detailed Description

Интерфейс прав доступа к экземплару объекта на основе настроек родительского объекта Пример 1: доступ к версии документа настраивается в карточке документа, правами могут обладать:

  • автор
  • пользователь
  • группа
  • эл-т организационной структуры
