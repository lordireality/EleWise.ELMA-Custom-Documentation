Detailed Description

Сервис для ограничения работы с зависимостями форм.

Зависимости форм создаются при сохранении метаданных. При импорте метаданных данный функционал нужно отключить. Именно этим и занимается данный сервис. Так же его можно будет использовать и при публикации метаданных, чтобы там отключить работу с зависимостями форм
