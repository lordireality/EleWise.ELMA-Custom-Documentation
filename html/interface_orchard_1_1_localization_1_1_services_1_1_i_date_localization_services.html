Detailed Description

Provides conversion and formatting of dates according to the Orchard configured time zone, culture and calendar (as opposed to the system configured time zone and culture).

Converts a localized date string to a UTC date in Gregorian calendar.

Parameters
dateStringThe localized date string to convert.
optionsAn optional DateLocalizationOptions instance used to control various aspects of the conversion process.
Returns
A DateTime instance where the time component equals that of DateTime.MinValue.

If the dateString parameter is null or equal to DateLocalizationOptions.NullText property, this method returns null.

DateTime? Orchard.Localization.Services.IDateLocalizationServices.ConvertFromLocalizedString ( string dateString,
string timeString,
DateLocalizationOptions options = null )

Converts separate localized date and time strings to a single UTC date in Gregorian calendar.

Parameters
dateStringThe localized date/time string to convert.
timeStringThe localized time string to convert.
optionsAn optional DateLocalizationOptions instance used to control various aspects of the conversion process.
Returns

If the dateString parameter is null or equal to DateLocalizationOptions.NullText property, the returned DateTime instance will have a date component that equals that of DateTime.MinValue. If the timeString parameter is null or equal to DateLocalizationOptions.NullText property, the returned DateTime instance will have a time component that equals that of DateTime.MinValue. If both dateString and timeString parameters are null or equal to DateLocalizationOptions.NullText property, this method returns null.

Implemented in Orchard.Localization.Services.DefaultDateLocalizationServices.
