enum EleWise.ELMA.CAB.Commands.CommandStatus

Status of a T:EleWise.ELMA.CAB.Commands.Command at any given time.

Enumerator
Enabled 

T:EleWise.ELMA.CAB.Commands.Command will fire the E:EleWise.ELMA.CAB.Commands.Command.ExecuteAction event when it is executed (either directly through M:EleWise.ELMA.CAB.Commands.Command.Execute or through an E:EleWise.ELMA.CAB.Commands.CommandAdapter.ExecuteCommand event in a registered adapter. Adapters should show the appropriate UI elements.

Disabled 

T:EleWise.ELMA.CAB.Commands.Command will not fire the E:EleWise.ELMA.CAB.Commands.Command.ExecuteAction event even if it is executed (either directly through M:EleWise.ELMA.CAB.Commands.Command.Execute or through an E:EleWise.ELMA.CAB.Commands.CommandAdapter.ExecuteCommand event in a registered adapter. Adapters should disable UI elements (or equivalent operation available for it), meaning that it will be visible but not enabled, in the general case.

Trying to execute a disabled command does not perform any action and does not raise any exceptions.

Unavailable 

T:EleWise.ELMA.CAB.Commands.Command will not fire the E:EleWise.ELMA.CAB.Commands.Command.ExecuteAction event even if it is executed (either directly through M:EleWise.ELMA.CAB.Commands.Command.Execute or through an E:EleWise.ELMA.CAB.Commands.CommandAdapter.ExecuteCommand event in a registered adapter. Adapters should hide UI elements (or equivalent operation available for it), meaning that it will be visible but not enabled, in the general case.

Trying to execute an unavailable command does not perform any action and does not raise any exceptions.
