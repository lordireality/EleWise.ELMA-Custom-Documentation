◆ AssertionConditionType

enum EleWise.ELMA.ComponentModel.AssertionConditionType

Specifies assertion type. If the assertion method argument satisfies the condition, then the execution continues. Otherwise, execution is assumed to be halted.

Enumerator
IS_TRUE 

Marked parameter should be evaluated to true.

IS_FALSE 

Marked parameter should be evaluated to false.

IS_NULL 

Marked parameter should be evaluated to null value.

IS_NOT_NULL 

Marked parameter should be evaluated to not null value.

enum EleWise.ELMA.ComponentModel.CollectionAccessType
Enumerator
None 

Method does not use or modify content of the collection.

Read 

Method only reads content of the collection but does not modify it.

ModifyExistingContent 

Method can change content of the collection but does not add new elements.

UpdatedContent

Enumerator
All

Server 

Компонент предназначен для использования в контексте сервера (в случаях запуска в рамках веб-приложения или дизайнера)

WebServer 

Компонент предназначен для использования в контексте сервера, запущенного в рамках веб-приложения

Web 

Компонент предназначен для использования в контексте веб-приложения

DesignServer 

Компонент предназначен для использования в контексте сервера, запущенного в рамках дизайнера (при локальном подключении к конфигурации)

Design 

Компонент предназначен для использования в контексте дизайнера (вне зависимости от типа подключения к серверу)

Test 

Компонент предназначен для использования в контексте тестирования
