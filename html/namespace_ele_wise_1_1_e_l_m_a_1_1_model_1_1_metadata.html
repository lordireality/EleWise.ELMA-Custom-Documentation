Сборки с этим статусом опубликованы и должны быть загружены при старте приложения

◆ CopyAction

enum EleWise.ELMA.Model.Metadata.CopyAction

Действие при копировании объекта

Enumerator
Default 

По умолчанию

Выбирается действие согласно контекста копирования

ByRef 

Всегда копировать по ссылке

Работает аналогично действию "Сохранять исходную ссылку", но если объект, на который указывает ссылка, уже скопирован (при общей операции копирования), то будет использоваться этот скопированный объект.

Clone 

Всегда создавать копию

Всегда будет создаваться клон объекта

Ignore 

Не копировать

Никогда не будет копироваться

SaveRef 

Сохранять исходную ссылку

При копировании будет сохраняться ссылка на исходный объект.
